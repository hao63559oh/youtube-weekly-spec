## 2. UI デザイン（確定事項）
- **配色**：ワインレッド基調（`--bg:#3a1015`）＋斜光グラデ。アクセントは深紅から浮く YouTube 赤（`--accent:#ff3127`）＋ワインレッドの差し色。
- **タイトル**：「**YouTube PickUp**」のみ（serif・左に赤バー）。サブタイトル削除、更新日時は小さく残置。
- **固定ヘッダー**：タイトル＋週／並び替え／タグ／エクスポートを1枚にまとめ `position:sticky` で追従（半透明＋ブラー）。
- **カード（参考カタログに合わせたレイアウト）**：
  - 全面サムネの**横長ボード（2:1）**を**1カラム**で縦に流す。
  - 奇数=左寄せ／偶数=右寄せの**左右ジグザグ**＋軽い重なり（参考の構図）。狭い画面では全幅縦積み。
  - 情報は**左上に半透明シェイプ＋グラデ**で重ね、操作系（YouTubeで開く／コピー）は右下ホバー、good/bad は右上ホバー。再生中はオーバーレイを隠す。
- ダーク基調に統一（OS ライト追従は一旦無効化）。

## 3. 追加した絞り込み機能
- **全期間横断ビュー**：週セレクタに「**すべて**」を追加 → `docs/data/all.json` を読み、全件を混在表示（各カードに**週バッジ**）。並び替え・タグが全件に効く。
- **タグ（ジャンル／ソース／評価を統一）**：
  - **よく使う順バー**（常時表示・上位6個）。使用回数を `localStorage(tagUsage:v1)` に記録、並び替えは週/モード切替時に反映。
  - **🔍 ボタンで全タグパネル**を開閉（ジャンル／ソース／評価をまとめて選択）。
  - **👍good / 👎bad もタグ**として絞り込み（`feedbackStore` と連携）。

## 4. データ層
- `scripts/fetch.py` に **`rebuild_all_json()`** を追加。週JSON群を唯一のソースに、index の週降順で全 videos を集約し各 video へ `week` 付与（冪等）。本処理（main）に組み込み済み。
- 出力：`docs/data/all.json`（`{updatedAt,count,videos[]}`）。
